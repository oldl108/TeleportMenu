# 传送菜单（Teleport Menu）· 国服 12.0.7 汉化版

原版作者：justw8（[CurseForge](https://www.curseforge.com/wow/addons/teleport-me-nu) / [GitHub](https://github.com/Justw8/TeleportMenu)）
本目录为**国服（魔兽世界 12.0.7 至暗之夜）完整汉化版**，所有界面文本与数据表注释均已中文化。

## 功能
把你的炉石、各类传送集成到游戏菜单（按 **ESC** 弹出的主菜单）里，一键使用：

- **炉石**：普通炉石 + 几十种炉石玩具（冬爷爷的炉石、无头骑士的炉石、啤酒节狂欢者的炉石、法夜/通灵领主/温西尔/格里恩盟约炉石、12.0 的光召炉石/菌语者的炉石孢子/核心守卫的炉石等）。
- **法师传送 / 传送门**：暴风城、铁炉堡、奥格瑞玛、达纳苏斯、达拉然（诺森德/破碎群岛/远古）、瓦德拉肯、多恩岛、银月城、至暗之夜的银月城等。
- **工程学虫洞**：诺森德、潘达利亚、阿古斯、库尔提拉斯、赞达拉、暗影界、巨龙群岛、卡兹阿加、奎尔萨拉斯等。
- **英雄之路（Hero's Path）**：各大版本地下城 / 团本传送（大地的裂变 → 熊猫人之谜 → … → 地心之战 → 至暗之夜）。
- **各职业传送**：萨满星界传送、武僧禅宗朝圣、死亡骑士死亡之门、德鲁伊梦境行走、狐人安营扎寨 / 返回营地、黑铁矮人地鼠机甲、哈兰尼尔根行等。
- **物品传送**：肯瑞托戒指系列、阵营披风、肯瑞托 / 夺日者信标、回城卷轴、各种传送玩具与勋章等。

## 安装
1. 把整个 `TeleportMenu` 文件夹复制到游戏插件目录：
   - 正式服：`World of Warcraft\_retail_\Interface\AddOns\TeleportMenu\`
   - 怀旧服：`World of Warcraft\_classic_\Interface\AddOns\TeleportMenu\`
2. 进入游戏，在角色选择界面左下角点「插件」，确保 **传送菜单（Teleport Menu）** 已勾选启用。
3. 如果你的客户端版本号与 `.toc` 里的 `## Interface: 120005, 120007` 不一致，请在插件列表里勾选「**加载过期插件**」。

## 使用
- 进入游戏后按 **ESC** 打开主菜单，右侧会出现一排炉石 / 传送按钮，鼠标移上去有提示，点一下即可使用。
- 输入 **`/tpm`** 打开设置面板（炉石偏好、图标大小、每行图标数、赛季传送开关等）。
- 输入 **`/tpm filters`** 打开物品过滤器（选择要显示 / 隐藏哪些传送物品）。

## 关于翻译说明
- **技能 / 物品名称**：插件通过法术 ID、物品 ID 直接调用游戏客户端数据，因此名称会**随你的客户端语言自动显示为中文**（国服即中文），无需额外翻译。
- **界面文本**：已完整汉化 `Locales/zhCN.lua`（共 165 条，与原版英文完全对应）。
- **数据表注释**：`Data/` 下的炉石、物品、虫洞列表里的英文注释已全部翻译为中文（保留英文原名便于对照）。
- 原版已内置简体中文（zhCN）语言文件，本汉化版在其基础上补全了少量遗漏字段，并将代码内注释、插件标题（`## Title-zhCN: 传送菜单`）一并中文化。
